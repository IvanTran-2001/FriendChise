The Roster tool is the weekly staffing board for FriendChise.

## What it shows

- Days run Mon–Sun as rows.
- Weeks run horizontally as columns.
- Each cell stores the members working that day, plus optional shift start and end times.
- Day configs add recommended staffing size and open/close windows.

## Templates

- Roster templates store repeating staffing patterns.
- A template can be stamped onto the live roster starting from a chosen Monday.
- That makes it useful for recurring weekly schedules and repeatable staffing plans.

## Why it matters

- It gives managers a quick way to see who works when.
- It keeps the live roster and reusable patterns separate.
- The shared schedule settings make it easier to keep staffing aligned with each org's operating hours.
